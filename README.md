# Frontend JS/TS Coding Lab

This folder contains small TypeScript practice files for strengthening core JavaScript concepts used in frontend interviews and daily React/Next.js work.

## Files

| File | Concepts covered |
| --- | --- |
| `array-methods.ts` | Callback functions, passing functions as arguments, async/await with Promises |
| `promises.ts` | Creating Promises, `resolve`, `reject`, `.then()`, and `.catch()` |
| `closures.ts` | Closures, private state, counter pattern, object methods using closed-over variables |

## How to run

From this folder:

```bash
cd /Users/jaspalsingh/Desktop/Saravo/practical-learning/frontend-js-ts-coding-lab
```

Run a file with `ts-node`:

```bash
npx ts-node closures.ts
npx ts-node promises.ts
npx ts-node array-methods.ts
```

If `ts-node` is not installed, install the required tools:

```bash
npm install -D typescript ts-node @types/node
```
